# picture_change_app

* swiftによるiosアプリ開発を学ぶために作成しました。  
* 画面右下のスタートボタンを押すと、画面上部の画像(５枚)が0.2秒づつの間隔で入れ替わっていくアプリです。
* 画面左下のストップボタンで、画像の入れ替わりがとまります。
